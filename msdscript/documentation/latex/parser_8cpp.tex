\doxysubsection{Detailed Description}
A tool to parse expressions written by the user into expressions that are usable by the script. 

Takes user written mathematical expressions and tokenizes them. Once the user input has been tokenized, the tokens are placed into expression objects and sent through the respective interp methods \begin{DoxyAuthor}{Author}
